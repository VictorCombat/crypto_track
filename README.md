## About the project

CryptoTrack is a cryptocurrency tracking application.
The purpose of this personal project is to learn how to develop a mobile application with a cross-platform framework.

<!-- BUILT WITH -->
## Built With

* [Flutter](https://flutter.dev/)
* [Dart](https://dart.dev/)
